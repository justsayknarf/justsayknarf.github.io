This site.

This site is being served with Jekyll, hosted on Github Pages.

This site is meant to be a fun platform for me to explore and learn. Continuing to add features bit by bit, pixel by pixel, page by page. I wanted to avoid Wordpress, Joomla, Squarespace, or any other CRM/website generator to be able to easily look behind the curtain without too much trouble, and be able to easily tweak without too much hand-holding.

The result is... pretty much what you're looking at. Various libraries, random tweaks here and there. All fun.
